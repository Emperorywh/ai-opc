2. **开始一个 Task**：把该行状态改为 🔄（in_progress）；开始前用 `git log` 校验与本表一致。
3. **遇阻**：状态改 ⚠️ blocked，备注写阻塞原因与需要的输入，**停下并问我**。
4. **绝不**跳过未 ✅ 的前置 Task；**绝不**修改本文件来"绕过"依赖。
